# Discovering Core Stocks

## Overview
核心股不是先猜出来的，而是在 **主线题材 → 统计节点 → 候选筛选 → 竞价/盘中确认** 里被市场筛出来的。回答这类问题时，必须先讲题材和节点，再讲个股和买点。

## When to Use
适用场景：
- 用户想从题材里找“核心股”或“龙头苗子”
- 用户想区分真核心、假核心、跟风股
- 用户要把复盘内容整理成盘前 / 盘中 checklist
- 用户给了一批帖子、复盘、题材名，想提炼成固定流程
- 常见触发词：题材核心股、核心股选股、龙头苗子、见股起意、复盘 checklist

不适用：
- 没有明确主线题材，只有零散个股
- 用户想做长线基本面选股
- 明显处于全面退潮 / 普杀期，却要求强行抄底找核心

## Required Output Structure
回答时固定按这个顺序输出，避免泛泛而谈：

1. **主线题材 / 市场环境**
2. **统计节点**
3. **候选筛选标准**
4. **真核心确认信号**
5. **假核心 / 淘汰信号**
6. **买点与离场**
7. **盘前 / 盘中 checklist**（如果用户要实战版）

## Core Workflow

### 1. 先锁定主线题材
- 只在主线里找核心，不在杂毛里猜答案。
- 大题材启动先看持续性；没有主线时，宁可少做。

### 2. 再找统计节点
优先看：
- 指数见底日
- 指数与题材共振爆发日
- 上涨家数冰点 / 连板冰点
- 前一高标断板日前后 1 天，尤其断板后第 2~3 天
- 新题材爆发首日

**必须明确说明：节点是“核心股出生的日子”。**

### 3. 在主线题材里筛候选
候选股至少从这些维度比较：
- 概念纯度
- 辨识度
- 带动性
- 涨幅 / 量能 / 封单
- 资金介入深度
- 抗跌性
- 股性 / 市值是否适合短线炒作

### 4. 用竞价和盘中确认核心
保留：
- 一字最强
- 弱转强
- 急杀后有承接
- 炸板后快速回封
- 能带动后排或帮助题材继续打高度

淘汰：
- 题材不匹配主线
- 只会讲故事、逻辑不硬
- 开盘直接被按且承接差
- 走得磨叽、不主动、锁不住筹码
- 错过最佳买点后只剩追高冲动

### 5. 再决定怎么参与
- 启动期：要信早信，先上车比抠完美买点更重要
- 爆发 / 主升期：竞价、回封、打板都可能是买点
- 分歧期：优先看深水急杀有承接的低吸，或炸板回封确认
- 错过最佳买点：不要硬追，等下一次分歧确认

### 6. 最后给离场标准
至少要提到这些见顶 / 离场信号：
- 高位连续加速 2~3 板
- 爆量大阴 / 大绿棒
- 跌停
- 连续 2~3 天不创新高
- 不成长就离场

## Quick Reference

| 维度 | 真核心特征 | 假核心 / 跟风特征 |
|---|---|---|
| 题材位置 | 主线题材前排 | 非主线、杂毛 |
| 逻辑 | 逻辑硬、能反复验证 | 讲故事、靠情绪 |
| 地位 | 领涨、抗跌、能带动后排 | 被动跟涨、带不动板块 |
| 竞价 | 一字、弱转强、主动性强 | 直接被按、开盘转弱 |
| 分时 | 急杀有承接、炸板能回封 | 走得磨叽、无承接 |
| 处理方式 | 分歧可低吸，见顶再走 | 错过不追，转弱即删 |

## Common Mistakes
- **先看个股，后看题材。** 正确顺序一定是先题材、后节点、再个股。
- **过早定义龙头。** 真龙头 / 真核心是市场走出来的，不是嘴上封的。
- **只看连板高度，不看题材匹配。** 连板高但不在主线，优先删。
- **错过最佳买点后追高。** 很多亏损不是买错，是追高后又加仓。
- **只讲买点，不讲淘汰条件。** 没有淘汰信号的答案不完整。
